# Raspberry-Pi-Powered-Autonomous-Navigation-Robot
AI Use & Authorship Declaration
Project Overview
This repository contains the technical implementation, data analysis, and documentation for the EEEE1027 Raspberry Pi Line-Following Robot project. In accordance with university academic integrity guidelines, this statement clarifies the role of Artificial Intelligence in the development of this work.

Authorship & Originality
The core engineering concepts, project architecture, and intellectual property within this project are the original work of the author. This includes, but is not limited to:

Experimental Design: The logic for the "Donut Hole" recycle symbol test, the "Bumper Slice" memory for shortcut navigation, and the aspect-ratio weighting for arrow detection.

Data Collection & Visualization: All raw experimental data was collected manually during lab sessions. All graphs and tables (including motor calibration curves and PID response charts) were generated by the author using original datasets.

Analytical Discussion: The critical interpretation of results, the comparison between theory and experiment, and the conclusions drawn regarding system stability.

Use of Generative AI Tools
Large Language Models (LLMs), specifically Gemini and Claude, were utilized as collaborative tools in the following capacities:

Code Implementation: Gemini and Claude were used to assist in the generation and debugging of specific Python routines. While the functional logic and control parameters (PID gains, HSV thresholds) were determined by the author through empirical testing, the AI assisted in syntactical structuring and library integration (OpenCV, RPi.GPIO).

Documentation & Structuring: Gemini was employed as a writing assistant to help organize and structure technical paragraphs in the logbook and final report. It assisted in refining the professional tone and ensuring the narrative flow met the required academic standards.

Verification
The author has reviewed, verified, and tested all AI-assisted outputs to ensure technical accuracy and adherence to the project's physical constraints. All final design decisions and tuning parameters were validated through real-world track testing.
